# Практическая работа

## Вариант №15

В процессе функционирования **АСУ ВД** в файле фиксируются данные о самолётах, совершивших посадку.

Каждая запись имеет структуру:

| Поле | Пример | Описание |
|---|---:|---|
| Время посадки | `14:25` | время посадки самолёта |
| Марка ЛА | `ИЛ-906М` | марка летательного аппарата |
| Бортовой номер | `Б-2387` | бортовой номер самолёта |
| Пункт отправления | `Тамбов` | пункт отправления |

## Задание

1. Подготовить программу, сортирующую записи с использованием **индексной сортировки методом «пузырька»** в порядке возрастания времени посадки.
2. Результаты печатать в виде таблицы.
3. Обеспечить входной контроль **времени посадки** и **бортового номера**.
4. Выполнить отладку и тестирование программы.

## Требования к реализации

- Чтение данных из файла выполнять с использованием функций ввода/вывода языка **C++**.
- Алгоритм должен быть параметризован.
- Обмен данными с подпрограммой должен осуществляться только через параметры.
- Исходные данные должны храниться в отдельном файле.
